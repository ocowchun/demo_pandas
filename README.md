## demo_pandas
> demo pandas feature with python3

## install python and pandas

### install pyenv
https://github.com/yyuu/pyenv

```bash
$ brew install pyenv
```

### install python 3.5
``` bash
$ pyenv install 3.5.0
$ pyenv global 3.5.0
$ pyenv rehash
```

### install pandas
```sh
$ pip install pandas
```
